to add file to staging area:
git add filename.txt
git add filename1.txt filename2.txt => two files
git add * => all files

To commit the changes:
git commit -m 'message'

Push our code to github/to move file from staging area to Github repository:
git remote add origin https://github.com/nabin1000/push_noti.git
git push origin master

git log => View commit history
git log --oneline => View in oneline

To go into specified version:
git checkout SHA1 code
git checkout master

To copy(clone) remote file into local:

git clone project_url
